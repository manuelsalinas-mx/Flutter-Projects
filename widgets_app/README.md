
# Widgets app

Contiene lo fundamental para cualquier aplicación de Flutter como:

1.- **Navegación entre pantallas**

2.-  **Nuevos widgets**
- Botones y sus variantes
- Botones personalizados
- Tarjetas
- Tarjetas personalizadas
- Align

3.-  **Rutas**
- Propias de Flutter
- Go_Router
- Paths
- Configuraciones de router
- - Propio
- - De terceros

4.- **Más Widgets**
- RefreshIndicator
- InfiniteScroll
- ProgresIndicators
- - Lineales
- - Circulares
- - Controlados
- Animaciones
- Snackbars
- Diálogos
- Licencias
- Switches, Checkboxes, Radios
- Tiles
- Listas
- Pageviews

Se dejan las bases necesarias y así comprender mejor como construir nuestras aplicaciones.
